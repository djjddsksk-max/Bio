# Frontend UI Improvements

## Задача
Улучши UI/UX фронтенда приложения Bio

## Что нужно сделать

### 1. Главная страница (Home)
- Добавь hero секцию с анимацией
- Создай секцию "Features" с карточками возможностей
- Добавь секцию "How it works" с пошаговым объяснением
- Сделай footer с контактами и ссылками

### 2. Страница Auth
- Улучши форму логина/регистрации
- Добавь валидацию в реальном времени
- Покажи ошибки красиво через toast уведомления
- Добавь "Forgot password?" функционал (UI только)
- Сделай переключение между Login/Register плавным

### 3. Dashboard
- Создай боковое меню (sidebar) с навигацией
- Добавь карточки со статистикой
- Сделай таблицу с данными пользователя
- Добавь графики (используй recharts)
- Создай секцию "Recent Activity"

### 4. Profile
- Форма редактирования профиля
- Загрузка аватара (UI только, без backend)
- Смена пароля
- Настройки уведомлений
- История активности

### 5. Общие улучшения
- Добавь темную/светлую тему (используй next-themes)
- Сделай адаптивный дизайн для мобильных
- Добавь loading состояния
- Улучши анимации (framer-motion)
- Добавь breadcrumbs для навигации

## Технологии
- React 19
- TypeScript
- TailwindCSS
- Radix UI компоненты (уже установлены)
- Framer Motion для анимаций
- Recharts для графиков

## Примечания
- Используй существующие UI компоненты из client/src/components/ui/
- Следуй современным UI/UX трендам
- Делай код чистым и переиспользуемым
